# RestoreMechacatOS
This tool lets you restore your MechacatOS instance. We use users as folders with their data in your system. A bug in your MechacatOS instance may have occured, which destroyed the folder. Do not worry, do not panic. You can click on the "MechaCat" icon, then click Files. Then you can create a folder with your new username. You will have to create a file named "spw.pw". PW stands for password, and SPW means Secure Password.
